# Vacunación por el Covid-19 en Argentina
___
### Preguntas:

* ¿Cual fue el mes donde vacunaron mas civiles?
###### Abril fue el mes donde mas dosis se le aplicaron a los civiles, mientras que Enero fue donde se dieron menos dosis.
* ¿Cual fue el rango de edad mas vacunado hasta la fecha?
###### Las personas de 51 a 60 años fueron las personas mas vacunas hasta la fecha
* ¿A que genero se le administraron mas dosis?
###### Las mujeres fueron mas vacunadas a comparacion de los hombres.
* ¿Que vacuna fue la mas aplicada?
###### Como podemos observar la vacuna Sputnik fue la mas usada con un 47,4%, mientras la menos utilizada fue la moderna con un 2,2%.
* ¿Que lugar, donde se aplicaron las vacunas, fue el mas utilizado?
###### De los 3 tipos de lugares que hay, para vacunar a las personas, los lugares publicos fueron los que mas se usaron.
___

### Conclusion
La conclusion que podemos obtener en base a las preguntas y las respuestas obtenidas son que las personas entre las edades de 51 a 60 años fueron las mas vacundas ya que esto se debera a que son de factor de riesgo. El mes donde mas dosis se aplicaron en las personas fue Enero (en lugares publicos), luego tenemos que priorizaron mas la vacunacion en las mujeres que en los hombres y que las vacunas mas utilizadas fueron las de Sputnik.

import numpy as np
import pandas as pd
import matplotlib.pyplot as plt1
import matplotlib.pyplot as plt2
import matplotlib.pyplot as plt3
import matplotlib.pyplot as plt4
import matplotlib.pyplot as plt5

df1=pd.read_csv('dataset_total_vacunas.csv')

def cambiarFechaAMes(valor):
    valor = str(valor)
    if valor.__contains__("APR"):
        return "APR"
    if valor.__contains__("AUG"):
        return "AUG"
    if valor.__contains__("FEB"):
        return "FEB"
    if valor.__contains__("JUL"):
        return "JUL"
    if valor.__contains__("JUN"):
        return "JUN"
    if valor.__contains__("MAR"):
        return "MAR"
    if valor.__contains__("MAY"):
        return "MAY"
    if valor.__contains__("JAN"):
        return "JAN"

df1['FECHA_ADMINISTRACION'] = df1['FECHA_ADMINISTRACION'].apply(cambiarFechaAMes)

yValuesDia = df1['FECHA_ADMINISTRACION'].value_counts()
xValuesDia = yValuesDia.keys()
yValuesDia = yValuesDia.values

#grafico1
plt1.plot(xValuesDia,yValuesDia,marker='o')
plt1.title('MESES DEL 2021')
'''plt1.xlabel('OPCIONES')
plt1.ylabel('PERSONAS QUE RESPONDIERON')'''
plt1.show()
